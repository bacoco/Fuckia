# Third Reviewers

Third reviewers start as read-only.

Templates:

- `third-model-plan-challenge`
- `third-model-code-challenge`
- `third-model-synthesis`

The third reviewer receives a bounded packet:

- Linear contract;
- plan;
- diff;
- target workflow;
- known forbidden files;
- exact output schema.

The third reviewer must not write files unless converted into a normal implementer issue with locks and gates.
