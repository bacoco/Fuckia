# Platform Adapters

Platform-only skills are thin adapters around shared governance.

Claude-only adapters:

- `claude-linear-native-runner`
- `claude-plan-reviewer`
- `claude-code-reviewer`
- `claude-subagent-guard`
- `claude-memory-export`

Codex-only adapters:

- `codex-linear-native-runner`
- `codex-pr-reviewer`
- `codex-subagent-guard`
- `codex-action-security`
- `codex-skill-plugin-packager`

Adapters may change invocation mechanics. They must not weaken shared laws or gates.
