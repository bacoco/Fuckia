# Shared Skills System

Shared and platform-only skills must enforce `docs/vibe-coding/constitution/agent-constitution.md`.

The constitution is not a skill. Skills are one enforcement mechanism for the constitution.

1. `shared/categories.md`

## Core Rule

Claude and Codex skill formats are not identical.

Use one neutral source and generate platform-specific outputs.

Platform-only skills may adapt mechanics. They must not weaken shared governance.
