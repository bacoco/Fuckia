# Existing Project Governance Auditor

`existing-project-governance-auditor` is a required skill for migrations.

## Purpose

It audits an existing project before Fuckia writes anything.

## It Must Read

- `AGENTS.md`;
- `CLAUDE.md`;
- `.agents/skills/...`;
- `.claude/skills/...`;
- `.github/workflows/...`;
- PR templates;
- local specs and docs;
- repository structure;
- active project conventions.

## It Must Produce

- governance inventory;
- conflict report;
- scattered-spec map;
- skill divergence report;
- proposed installation plan;
- files that will be touched;
- files that must not be touched;
- residual risks.

## It Must Not Do

- modify product code;
- delete existing docs;
- overwrite existing agent rules;
- install strict mode;
- reorganize the repository without an approved plan.
