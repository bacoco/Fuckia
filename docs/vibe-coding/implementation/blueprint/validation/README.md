# Validation

Strict mode blocks:

- missing constitution references in generated skills/templates;
- missing, divergent, or weakened role law propagation;
- generated skill drift;
- forbidden file edits;
- destructive-change guard excess;
- missing Linear ID;
- risky work without independent plan-review;
- missing verification receipt for product/UI work;
- self-review on risky work without human bypass;
- unsupported uncertain causal language in required plans, reviews, receipts, and handoffs.

Warning mode reports findings without blocking.
