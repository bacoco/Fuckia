# Docs

This directory contains every long-form Fuckia document. Root-level folders are reserved for source code, templates, scripts, tests, and package metadata.

## Directory Map

- `fuckia/` - governance files installed into this repository so Fuckia follows its own rules.
- `vibe-coding/` - constitution, operating model, skills design, installation design, and implementation history.
- `agent-runbooks/` - one-prompt runbooks for Claude Code and Codex.
- `examples/` - generated examples, including generated Claude and Codex skills.
