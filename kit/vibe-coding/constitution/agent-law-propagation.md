# Agent Law Propagation

The system uses layered laws.

The constitution is universal. Role-specific rules are addenda. Validators enforce both.

## Sections

1. `propagation/layers.md`
2. `propagation/role-addenda.md`
3. `propagation/generated-skill-requirements.md`
4. `propagation/validators.md`
5. `propagation/fixtures.md`

## Core Rule

Do not choose between universal laws and validator/developer rules. Implement both:

- universal constitution for every agent;
- role-specific addenda for each workflow role;
- deterministic validators that fail missing, divergent, or weakened laws.
