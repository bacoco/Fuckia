# Kit

Fuckia implementation files live here.

Directory map:

- `src/` - CLI source code.
- `templates/` - files installed into target repositories.
- `skills-src/` - neutral shared skill sources.
- `generated-skills/` - generated Claude and Codex skill previews.
- `agent-runbooks/` - one-prompt agent runbooks.
- `vibe-coding/` - design notes, constitution, and operating model.
- `tests/` - CLI tests.
- `scripts/` - build scripts.
