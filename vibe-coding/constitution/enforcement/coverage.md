# Coverage

The laws must appear in:

- root `README.md`;
- generated Claude skills;
- generated Codex skills;
- `AGENTS.md`;
- `CLAUDE.md`;
- Linear templates;
- GitHub PR templates;
- GitHub validators;
- plan-review and code-review gates;
- verification receipts.

Automatically checkable coverage must be enforced by deterministic validators.

Coverage that cannot be checked automatically must be represented by explicit checklist gates with actor identity and receipt.
