# Done And Ready

## Definition Of Ready

`implement` is Ready only when:

- Linear contract is complete;
- file locks are declared;
- delete budget is declared;
- existing pipeline is named or explicitly not applicable;
- required plan-review is approved or human-waived;
- implementer has posted pre-flight.

## Definition Of Done

Done requires a verification receipt:

```text
Linear issue:
PR:
commit SHA:
checks run:
target environment:
real route/workflow tested:
evidence:
result:
residual risks:
```

Typecheck and unit tests are not Done by themselves.
