# Implementation

This directory contains the implementation study and standalone repo plan.

Read order:

1. `repo-template.md`
2. `capabilities-decisions-and-open-risks.md`
3. `capabilities/README.md`
4. `blueprint/README.md`
5. `rollout-plan.md`

## Purpose

These files define how to build the standalone collaboration kit without embedding it in any product repository.

Implementation must start in warning mode, prove the blank repo demo, then run one existing-project pilot before strict rollout.
